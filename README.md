# PORTFOLIO
# 💻 Portfolio Développeur

## 📌 Présentation du projet

Ce projet est un site web portfolio statique contenant quatre pages principales: Accueil, A propos, Mes projets et Contact. La navigation est assure par une barre de menu (navbar) presente sur toutes les pages. 

---

## 🎯 Sujet choisi

**Portfolio Développeur Web** — Création d'un site portfolio statique pour présenter nos compétences techniques, nos projets et nos parcours en développement web durant notre cursus a l'UCAK.

---

## 🛠️ Technologies utilisées

| Technologie | Utilisation |
|---|---|
| HTML5 | Structure des pages |
| CSS3 | Mise en forme et styles |
| JavaScript | Interactions et dynamisme |
| Bootstrap / Tailwind CSS | Design responsive |
